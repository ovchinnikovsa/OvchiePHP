# Blog

Just dummy pet.

###  Requirements:
* linux or win, under wsl
* installed docker
* installed make app

### Steps to build

Run make instructions:

1. `make build`
2. `make run`
3. `make db_restore`

To stop - `make stop`

> Also u can change db access in /docker/conf/db.ini file. And conf.ini from /src with salt.
